# 快速开始

```shell

nrm use xxx

npm i
npm run build

```

## 发布

先改好版本号,再执行发布命令便可。

```shell
cd ./
nrm use xxx
npm publish

```

## 测试

```shell
npm run test
npm run test:coverage
```
